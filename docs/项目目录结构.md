# 项目目录结构

## 仓库概览

```
ZiroCode/
├── server/                 # Fastify 后端 (:4000)
├── frontend/               # React 19 + Vite 前端 (:5173)
├── nginx/                  # Nginx 反向代理配置
├── docker-compose.yml      # 5个服务编排
└── docs/                   # 项目文档
```

---

## server — Fastify 后端

```
server/
├── src/
│   ├── app.ts              # 入口文件，注册插件和路由
│   ├── routes/             # API 路由
│   │   ├── auth.ts         # 认证（注册/登录/登出/用户信息）
│   │   ├── keys.ts         # API Key 管理（CRUD）
│   │   ├── subscriptions.ts # 订阅管理（列表/兑换）
│   │   ├── user.ts         # 用户数据（仪表板/配额/统计）
│   │   ├── v1.ts           # AI 网关（OpenAI 兼容）
│   │   └── health.ts       # 健康检查 + 模型渠道列表
│   └── lib/                # 共享工具库
│       ├── auth.ts         # JWT 生成/验证
│       ├── db.ts           # Prisma 客户端单例
│       ├── quota.ts        # 配额计算/扣减/限流/API Key生成
│       └── api-utils.ts    # 认证中间件/错误处理
├── prisma/
│   ├── schema.prisma       # 7张表数据模型
│   └── seed.ts             # 种子数据脚本
├── .env                    # 环境变量
├── Dockerfile
├── package.json
└── tsconfig.json
```

---

## frontend — React 19 + Vite 前端

```
frontend/
├── src/
│   ├── main.tsx            # 入口（TanStack Query + Router）
│   ├── App.tsx             # 路由配置
│   ├── index.css           # 全局样式（Tailwind）
│   ├── pages/              # 页面组件
│   │   ├── LoginPage.tsx   # 登录页
│   │   ├── RegisterPage.tsx # 注册页
│   │   ├── DashboardPage.tsx # 仪表板
│   │   ├── KeysPage.tsx    # API密钥管理
│   │   ├── SubscriptionPage.tsx # 订阅管理
│   │   └── UsagePage.tsx   # 使用统计
│   └── components/         # 共享组件
│       ├── AppLayout.tsx   # 布局 + 导航栏
│       └── ReferralSection.tsx # 邀请推广
├── index.html
├── vite.config.ts          # Vite配置 + API代理
├── Dockerfile
└── package.json
```

---

## 服务端口

| 服务 | 端口 | 说明 |
|------|------|------|
| Nginx | 3000 | 统一入口 |
| Fastify 后端 | 4000 | API 服务 |
| React 前端 | 5173 | 开发服务器 |
| PostgreSQL | 5432 | 数据库 |
| Redis | 6379 | 缓存/限流 |
