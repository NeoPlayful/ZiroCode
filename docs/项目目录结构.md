# 📁 项目目录结构

## 仓库概览

```
ZiroCode/
├── src/
│   ├── web/              # Next.js 主应用（前端 + Route Handlers）
│   ├── server/           # AI 网关服务（Node.js + Fastify）
│   └── shared/           # 共享类型定义
├── docker-compose.yml
└── package.json          # workspace root
```

---

## src/web — Next.js 主应用

```
src/web/
├── app/                      # App Router 页面
│   ├── (auth)/               # 认证页面组（不含布局）
│   │   ├── login/
│   │   ├── register/
│   │   └── reset-password/
│   ├── (dashboard)/          # 主应用页面组（含侧边栏布局）
│   │   ├── layout.tsx
│   │   ├── page.tsx          # 仪表板
│   │   ├── keys/
│   │   ├── models/
│   │   ├── referral/
│   │   ├── subscription/
│   │   ├── tickets/
│   │   └── settings/
│   └── api/                  # Route Handlers
│       ├── auth/
│       ├── user/
│       ├── keys/
│       ├── models/
│       ├── notifications/
│       ├── referral/
│       ├── subscriptions/
│       ├── tickets/
│       ├── payment/
│       ├── analytics/
│       └── announcements/
├── components/               # React 组件
│   ├── common/
│   ├── dashboard/
│   ├── keys/
│   ├── models/
│   ├── referral/
│   ├── subscription/
│   └── tickets/
├── lib/                      # 工具库（db、auth、redis 等）
├── hooks/                    # 自定义 React Hooks
├── stores/                   # Zustand 状态
├── types/                    # TypeScript 类型定义
├── i18n/                     # 国际化配置与翻译文件
├── public/                   # 公开静态文件
├── .env.example

└── prisma/

```

## src/server — AI 网关服务

```
src/server/
├── src/
│   ├── routes/           # Fastify 路由
│   │   └── proxy.ts      # AI 请求代理转发
│   ├── plugins/          # Fastify 插件（限流、认证等）
│   ├── middleware/       # 请求中间件
│   └── index.ts          # 入口文件
├── .env.example
├── Dockerfile
├── package.json
└── tsconfig.json
```
