# 项目目录结构

## 仓库概览

```
ZiroCode/
├── server/                 # Fastify 后端 (:4000)
├── frontend/               # React 19.2.6 + Vite 前端 (:5173)
├── nginx/                  # Nginx 反向代理配置
├── docker-compose.yml      # 5个服务编排
└── docs/                   # 项目文档
```

---

## server — Fastify 后端

```
server/
├── src/
│   ├── app.ts              # 入口文件，注册插件和路由
│   ├── routes/             # API 路由
│   │   ├── auth.ts         # 认证（注册/登录/登出/密码重置/邮箱验证）
│   │   ├── user.ts         # 用户数据（仪表板/配额/统计/个人资料）
│   │   ├── keys.ts         # API Key 管理（CRUD + 使用统计）
│   │   ├── subscriptions.ts # 订阅管理（列表/兑换/购买/套餐）
│   │   ├── referral.ts     # 推荐系统（统计/领取/分析/排行榜）
│   │   ├── tickets.ts      # 工单系统（CRUD/回复/搜索/导出/分类/模板）
│   │   ├── payments.ts     # 支付系统（Stripe结账/Webhook/历史）
│   │   ├── analytics.ts    # 统计分析（使用量/成本/模型分布/导出）
│   │   ├── announcements.ts # 公告系统（列表/已读标记）
│   │   ├── notifications.ts # 通知系统（列表/已读/偏好设置）
│   │   ├── admin.ts        # 管理后台（用户/订阅/兑换码/渠道/工单/审计）
│   │   ├── webhooks.ts     # Webhook管理（CRUD/测试/日志）
│   │   ├── system.ts       # 系统配置（配置/教程）
│   │   ├── health.ts       # 健康检查 + 模型渠道列表
│   │   └── v1.ts           # AI 网关（OpenAI 兼容接口）
│   └── lib/                # 共享工具库
│       ├── auth.ts         # JWT 生成/验证
│       ├── db.ts           # Prisma 客户端单例
│       ├── quota.ts        # 配额计算/扣减/限流/API Key生成
│       ├── api-utils.ts    # 认证中间件/错误处理
│       ├── router.ts       # AI请求路由/负载均衡/健康检查
│       ├── stripe.ts       # Stripe支付集成
│       ├── email.ts        # 邮件发送（密码重置/验证）
│       ├── cache.ts        # Redis缓存封装
│       ├── notification.ts # 通知创建/推送
│       ├── referral.ts     # 推荐奖励计算/触发
│       ├── referral-fraud.ts # 推荐防作弊检测
│       ├── ticket-sla.ts   # 工单SLA计算/违规检测
│       ├── webhook-dispatcher.ts # Webhook事件分发
│       └── openapi.ts      # OpenAPI文档生成
├── prisma/
│   ├── schema.prisma       # 7张表数据模型
│   └── seed.ts             # 种子数据脚本
├── .env                    # 环境变量
├── Dockerfile
├── package.json
└── tsconfig.json
```

## 服务端口

| 服务 | 端口 | 说明 |
|------|------|------|

| React 前端 | 5173 | 开发服务器 |
| PostgreSQL | 5432 | 数据库 |
| Redis | 6379 | 缓存/限流 |
