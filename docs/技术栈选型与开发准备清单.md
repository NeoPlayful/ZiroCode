## 📋 一、项目实现前准备清单

### 1.1 技术栈选型

| 层级 | 推荐选型 |
|------|----------|
| 前端框架 | Next.js App Router + React |
| 状态管理 | Zustand / TanStack Query |
| UI 样式 | Tailwind CSS + shadcn/ui |
| 图表库 | echarts-for-react |
| 表单校验 | React Hook Form + Zod |
| 服务端 API/BFF | Route Handlers / Server Actions |
| ORM | Prisma / Drizzle |
| 数据库 | PostgreSQL |
| 缓存/限流 | Redis |
| 队列 | BullMQ |
| 认证 | Auth.js / 自研 JWT Cookie |
| 支付 | Stripe |
| 邮件 | Resend / AWS SES |
| AI 网关 | 独立 Node.js/Fastify 或 Go 服务 |
| 部署 | Docker + Nginx/Cloudflare / Vercel |

**AI 集成选项**
- CodeX API — 官方代码助手
- ClaudeCode API — 高级代码分析
- OpenAI API — 通用模型接口
