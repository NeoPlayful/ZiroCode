## 一、技术栈选型

| 层级 | 选型 | 版本 | 说明 |
|------|------|------|------|
| 前端框架 | React | 19 | Vite 构建 |
| 后端框架 | Fastify | 5 | Node.js 高性能框架 |
| 语言 | TypeScript | ~5 | 全栈类型安全 |
| 样式 | Tailwind CSS | 4 | Utility-first |
| 状态管理 | TanStack Query | 5 | 服务端数据缓存 |
| 路由（前端） | react-router-dom | 7 | 客户端路由 |
| 图标 | Heroicons | 2.2 | 统一图标库 |
| 图表 | ECharts + echarts-for-react | 最新 | 统计展示 |
| ORM | Prisma | 5.22 | PostgreSQL ORM |
| 数据库 | PostgreSQL | 16 | 主存储 |
| 缓存/限流 | Redis + ioredis | 7 / 5 | 限流计数器 |
| 认证 | 自研 JWT + jose | 6 | httpOnly cookie |
| 密码 | bcryptjs | 3 | 加密存储 |
| 反向代理 | Nginx | latest | 统一端口 |
| 部署 | Docker Compose | - | 5个服务 |

## 二、项目架构

```
ZiroCode/
├── server/                    # Fastify 后端 (:4000)
│   ├── src/
│   │   ├── app.ts            # 入口
│   │   ├── routes/           # 6个路由文件，15个API端点
│   │   └── lib/              # 4个工具库
│   ├── prisma/               # 数据库 schema + seed
│   └── Dockerfile
├── frontend/                 # React 19 + Vite (:5173)
│   ├── src/
│   │   ├── pages/            # 6个页面
│   │   ├── components/       # 共享组件
│   │   ├── App.tsx           # 路由配置
│   │   └── main.tsx          # 入口
│   └── Dockerfile
├── nginx/
│   └── default.conf          # 反向代理
├── docker-compose.yml        # 5个服务
└── .env.example
```

**端口规划：**
- `:3000` — Nginx 统一入口
- `:4000` — Fastify 后端
- `:5173` — React 开发服务器
- `:5432` — PostgreSQL
- `:6379` — Redis

## 三、开发环境设置

```bash
# 开发环境（本地）
cd server && npm run dev    # 启动后端 :4000
cd frontend && npm run dev  # 启动前端 :5173

# 生产环境
docker compose up -d        # 启动全部服务 :3000
```

## 五、安全措施

- ✅ 密码 bcrypt 加密存储
- ✅ JWT httpOnly cookie（防 XSS）
- ✅ API Key 格式 sk-xxx（防暴力）
- ✅ Redis 限流（30次/分钟）
- ✅ API 统一错误响应
- ✅ CORS 限制
