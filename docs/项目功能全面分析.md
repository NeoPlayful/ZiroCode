# 📊 项目功能全面分析

## 一、核心功能模块

### 1. 仪表板（Dashboard）

- 配额使用概览
- 按量额度显示（示例：464,642,496）
- 月卡额度显示（每天早上 8 点自动重置）
- 活跃订阅状态（1 个套餐）
- 订阅状态：永久有效

### 2. 兑换订阅

- 订阅套餐兑换
- 支持兑换码系统

### 3. 使用统计

- API 调用统计
- 消费记录查询
- 使用量分析

### 4. API 密钥管理

- 创建/删除 API Key
- 密钥权限管理
- 使用量监控

### 5. 模型广场

集中查看各渠道可用额度、支持模型和接入教程。

- 各渠道可用额度总览
- 支持模型列表展示
- 点击渠道按钮或下滑查看对应渠道接入教程

### 6. 工单中心

- 提交技术支持工单
- 工单状态跟踪
- 客服沟通

## 二、推荐系统

**邀请推广功能：**

- 邀请链接生成
- 邀请人数统计
- 已领取奖励记录
- 待领取奖励记录
- 收益记录查看
- 申请记录管理
- 奖励机制：好友兑换消费后获得奖励
